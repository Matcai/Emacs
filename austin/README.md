# Emacs 配置文件使用说明


### 使用方法
  * 将austin目录以及上一级目录的init.el文件复制到emacs.d/目录下即可。

###  at-latex
  * at-latex.el需要额外安装latex软件包以支持调用。
  * [MAC] 安装使用latexit
  * [Linux]  安装使用xelatex
  * [Windows] 安装使用xelatex

### at-packages
  * 添加新的packages 可以在jon/packages中新起一行加入packages名字重启emacs即可
  
### at-function
  * 主要用于添加函数实现相关功能
  
### at-keybinding
  * 存放按键绑定相关的配置
  
### at-setting
  * emacs 相关的初始化配置，环境变量的配置等。
